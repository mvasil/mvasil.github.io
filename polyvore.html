Overview

The Polyvore Outfits Dataset is a large-scale fashion dataset constructed from real user-created outfits on the Polyvore platform. It was designed to support research on visual compatibility, multimodal reasoning, and representation learning beyond simple similarity.

The dataset has been used in multiple papers across computer vision, recommendation systems, and multimodal learning. However, it is frequently referenced without proper attribution or with incorrect citations. This page serves as the authoritative reference for the dataset.

Dataset Description

  • User-created outfits composed of heterogeneous fashion items
  • Item images, categories, and outfit groupings
  • Designed for compatibility prediction, fill-in-the-blank, and retrieval tasks
  • Supports both similarity and compatibility reasoning

Intended Use

This dataset is intended for research in:

  • Fashion compatibility modeling
  • Visual recommendation systems
  • Metric learning and representation learning
  • Multimodal reasoning
