HandsOff: Labeled Dataset Generation with No Additional Human Annotations
Austin Xu, Mariya I. Vasileva, Achal Dave, Arjun Seshadri
CVPR, 2023
project page / code / arXiv

HandsOff is a framework capable of producing synthetic images with corresponding pixel-level labels without requiring additional human annotations. HandsOff utilizes a small number of real labeled images and exploits the rich latent representations of GANs to train an effectively infinite label generator.
